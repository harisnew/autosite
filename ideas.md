# Auto Garant — Дизайн-идеи

## Идея 1: «Тёмный Премиум» — Automotive Dark Luxury

<response>
<text>
**Дизайн-движение:** Dark Luxury / Automotive Premium — вдохновлён дилерскими сайтами премиум-класса (Porsche, Mercedes).

**Основные принципы:**
- Тёмный фон с контрастными акцентами создаёт ощущение премиальности
- Минимализм в деталях, максимализм в типографике
- Плавные анимации при скролле создают кинематографический эффект
- Большие визуальные блоки с авто на тёмном фоне

**Цветовая философия:**
- Основной: глубокий тёмно-серый (#0D0D0D) — элегантность и серьёзность
- Акцент: тёплый золотой (#D4A853) — премиальность, доверие
- Вторичный: холодный серый (#8A8A8A) — баланс и нейтральность
- Текст: белый и светло-серый — контраст и читаемость

**Парадигма макета:** Полноэкранные секции с параллакс-эффектом, вертикальный скролл с «snap» эффектом.

**Фирменные элементы:**
- Горизонтальные золотые линии-разделители
- Крупные числа (4%, 30%, 18) как декоративные элементы

**Философия взаимодействия:** Плавные hover-эффекты, элементы появляются при скролле с fade-in.

**Анимации:** Parallax на hero-секции, fade-in при скролле, hover-scale на кнопках.

**Типографика:** Montserrat Bold для заголовков + Roboto для текста.
</text>

## Идея 2: «Чистый Драйв» — Modern Glassmorphism

<response>
<text>
**Дизайн-движение:** Glassmorphism / Neo-brutalism hybrid — современный, свежий, технологичный.

**Основные принципы:**
- Стеклянные карточки с blur-эффектом поверх ярких градиентных фонов
- Асимметричная сетка с динамичными углами
- Крупная, уверенная типографика с акцентом на числах
- Яркие CTA-кнопки, которые «выпрыгивают» из интерфейса

**Цветовая философия:**
- Основной фон: глубокий индиго (#1A1A2E) — технологичность
- Акцент: электрический синий (#00D4FF) — энергия, движение
- Вторичный: мягкий фиолетовый (#6C63FF) — современность
- Стекло: rgba(255,255,255,0.08) с backdrop-blur

**Парадигма макета:** Асимметричная сетка, карточки «плавают» на разных уровнях глубины.

**Фирменные элементы:**
- Стеклянные карточки с размытым фоном
- Неоновые акценты на ключевых элементах

**Философия взаимодействия:** Карточки реагируют на hover с 3D-tilt эффектом, кнопки с glow.

**Анимации:** Floating elements, tilt-on-hover, staggered entrance animations.

**Типографика:** Space Grotesk для заголовков + DM Sans для текста.
</text>

## Идея 3: «Тёплое Доверие» — Warm Trustworthy Minimal

<response>
<text>
**Дизайн-движение:** Scandinavian Warm Minimalism — чистый, тёплый, вызывающий доверие.

**Основные принципы:**
- Светлый, тёплый фон с мягкими тенями создаёт ощущение надёжности
- Округлые формы и мягкие переходы — дружелюбность без «банковского» холода
- Акцент на читаемости и простоте — пользователь сразу понимает условия
- Крупный калькулятор как центральный элемент страницы

**Цветовая философия:**
- Основной фон: тёплый кремовый (#FAF8F5) — уют, доверие
- Акцент: насыщенный тёмно-зелёный (#1B4332) — стабильность, рост, деньги
- Вторичный: тёплый терракотовый (#C45B28) — энергия, действие
- Нейтральный: тёплый серый (#6B6560) — баланс

**Парадигма макета:** Широкие секции с большим padding, чередование фоновых блоков (кремовый / белый / зелёный), асимметричное расположение текста и изображений.

**Фирменные элементы:**
- Мягкие скруглённые карточки с тенями
- Иконки в стиле line-art с зелёным акцентом

**Философия взаимодействия:** Мягкие hover-переходы, кнопки с subtle scale, калькулятор с живым обновлением.

**Анимации:** Smooth scroll, fade-up при появлении секций, калькулятор с animated counter для чисел.

**Типографика:** DM Serif Display для заголовков (элегантность + доверие) + Inter для текста (читаемость).
</text>

## Выбранный подход: Идея 1 — «Тёмный Премиум»

Выбираю первый подход — тёмный премиальный дизайн. Он лучше всего подходит для автомобильной тематики, создаёт ощущение серьёзности и надёжности частного кредитора, а крупные числа условий (4%, 30%, 18 мес.) будут эффектно смотреться на тёмном фоне с золотыми акцентами.

**Шрифты:** Montserrat (заголовки) + Roboto (текст)
**Тема:** Dark
**Акценты:** Золотой (#D4A853) на тёмном (#0D0D0D)
